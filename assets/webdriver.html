Introduction

The idea behind developing this app is to build a pocket reference for Webdriver. Understanding the Webdriver concepts and how to set up the project.

Through out our learning we will take https://the-internet.herokuapp.com/ as test application.

After going through the App you should be able to get clarity on the following:

  • Webdriver concepts
  • Element Identification technique by XPath, CSS
  • Automating complex web elements
  • Understanding BDD using Cucumber with TestNG
  • Creation of Automation framework from scratch
  • Understanding the Page Object Model (POM) Framework
